# Contributing to FastEdit

Thanks for your interest in FastEdit!

## Bug Reports & Feature Requests

Please open an [issue](https://github.com/Orcaryn/fastedit/issues) with:

- **Bug reports** — steps to reproduce, browser/OS, and the input file format.
- **Feature requests** — describe the use case, not just the solution.

## Security

If you discover a security vulnerability, please report it responsibly. See [SECURITY.md](SECURITY.md) for details.

## Questions & Discussion

For questions about using FastEdit, check the [guides](https://fastedit.net/guides) or open a discussion.
